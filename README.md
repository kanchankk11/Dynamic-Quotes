## API Reference

#### Get weather info of a city

```http
    api.openweathermap.org/data/2.5/weather?q=${cityName}&units=metric&appid= <Your API Key>

```

| Parameter  | Type     | Description                    |
| :--------- | :------- | :----------------------------- |
| `cityName` | `string` | **Required**. Name of the city |
